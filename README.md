

## Сервер
### Зависимости
- сервер с s3
- linux + docker + java(в зависимости от версии сервера)
### Запуск
```
git clone https://github.com/launcherdev11/16Launcher-Server.git
cd 16Launcher-Server
mkdir minecraft
```
В папку minecraft положите ваш сервер, настройте .env
```
docker compose up -d
```
